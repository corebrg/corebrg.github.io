Java

CeMS는 Java (JVM) 기반에서 동작하기 때문에 우선 Java가 설치되어 있어야 합니다.
Oracle이 Java를 유료화 함에 따라 그의 대안인 무료 버전 OpenJDK를 사용할 수 있는지 고민 하는 관리자들이 많습니다. 결론적으로 말하자면 유료 Oracle Java 역시 OpenJDK를 바탕으로 고도화 된 산물이기 때문에 "사용하는데 문제가 없다" 입니다.

자바가 설치되어 있는지 확인하기 위해 명령 프롬프트에서 다음 명령어를 실행합니다.
그 전에 java 명령을 사용할 수 있도록 시스템 환경설정의 PATH 에 java bin 디렉토리가 포함되어 있어야 합니다.

java - version

openjdk version "12.0.1" 2019-04-16
OpenJDK Runtime Environment (build 12.0.1+12)
OpenJDK 64-Bit Server VM (build 12.0.1+12, mixed mode, sharing)

이와 같이 설치된 버전을확인할 수 있고, 설치되어 있지 않다면 명령이 수행되지 않을 것입니다.

Java 버전의 명명은 1.5 이후로 소숫점을 생략하고 6, 7, 8 과 같은 정수 형식으로 변경 되었고 최신 버전은 13 이며 14 버전이 준비중입니다
CeMSFree 는 2019년10월9일 기준 12 버전으로 컴파일 되었기 때문에 12 이상의 Java 환경이 필요합니다.
만일 실행 시 java.lang.UnsupportedClassVersionError 가 발생하고 class file version 에 대한 내용이 존재 한다면 아래 테이블을 참고하십시오.
